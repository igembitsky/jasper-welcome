Install Developer Tools

First, open Terminal on your Mac. Press ⌘ + Space to open Spotlight, then type "Terminal".

Terminal
🖥️
Terminal
Application — /System/Applications/Utilities
Hit Enter to open Terminal

Now paste these commands one at a time to install Homebrew (a Mac package manager) and Git (version control).

Terminal — zsh — 80×24
~ $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
==> Installation successful!
==> Next steps: Run brew help to get started
 
~ $ brew install git
==> Downloading https://ghcr.io/homebrew/core/git
✓ git 2.44.0 installed successfully
 
~ $ git --version
git version 2.44.0
